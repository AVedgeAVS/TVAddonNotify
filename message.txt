AVEdge HUB has a new update. Please install the latest update. The latest update
has the ADULT section on the Main Menu. It is passcode protected. The code is 999.
The lastest version is v4.1.0, updated 7-Apr-2017. Please run the AVEdge Updater 
app.

To check your current version; inside AVEdge HUB, go to SYSTEM, then scroll down to 
SERVICES and check the DEVICE NAME.

To get the latest update: First quit the AVEdge HUB app, then open the App Draw and 
click on the AVEdge Updater app. Click 'Continue', then choose AVEdge HUB Build, click 
'I Agree' and wait for the new version to download and install.
